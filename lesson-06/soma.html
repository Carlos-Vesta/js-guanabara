Conversão dos Tipos de Dados

* Conversão de dados do tipo String to Number

Para converter uma string para um número, usamos as seguintes instruções:


Ex1: var numero1 = Number.parseInt (window.prompt('Digite o primeiro número: '));

Ex2: var numero2 = Number.parseInt (window.prompt('Digite o segundo número: '));


Se os números forem decimais, iremos usar a seguinte instrução:

Ex1: var numero1 = Number.parseFloat (window.prompt('Digite o primeiro número: '));

Ex2: var numero2 = Number.parseFloat (window.prompt('Digite o segundo número: '));


Actualmente é possível usar apenas o Number que o sistema se encarregará de fazer a conversão para o seu devido tipo.

Ex2: var numero = Number (window.prompt('Digite um número: '));


NOTA: O comando window.prompt retorna uma String.
